# React Implementation — Law Firm Intake Triage

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for law firms.
- `src/views/PracticeAreaDemo.tsx` — Demo inquiry classification.
- `src/views/IntakeQualityCalculator.tsx` — Calculator for staff/attorney time saved.
- `src/views/FirmAuditForm.tsx` — Firm intake workflow audit form.
- `src/views/PilotSignup.tsx` — Pilot signup page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
