# React Implementation — B2B Podcast Repurposing System

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for founders/content teams.
- `src/views/BeforeAfterDemo.tsx` — Demo transcript to content transformation.
- `src/views/TranscriptUploadDemo.tsx` — Transcript paste/upload demo.
- `src/views/ContentCalendarPreview.tsx` — Calendar preview page.
- `src/views/EpisodeAuditForm.tsx` — Episode/source audit form.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
