# Admin Routes

- `/admin/b2b-podcast-repurposing-system/contentdashboard` — Sources, drafts, approvals and publishing queue.
- `/admin/b2b-podcast-repurposing-system/sourcelibrary` — Transcripts/notes with extracted themes and status.
- `/admin/b2b-podcast-repurposing-system/draftworkspace` — Edit drafts, regenerate sections and compare variants.
- `/admin/b2b-podcast-repurposing-system/voicesettings` — Audience, tone, banned phrases, examples and CTA preferences.
- `/admin/b2b-podcast-repurposing-system/calendarboard` — Scheduled/approved/published content by channel.
