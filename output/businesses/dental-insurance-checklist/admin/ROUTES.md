# Admin Routes

- `/admin/dental-insurance-checklist/verificationdashboard` — Pending verifications, completed checks and rework indicators.
- `/admin/dental-insurance-checklist/checklistqueue` — Patients/procedures awaiting verification steps.
- `/admin/dental-insurance-checklist/checklistdetail` — Checklist, source notes, completion state and staff comments.
- `/admin/dental-insurance-checklist/proceduretemplates` — Configure checklist templates by procedure category.
- `/admin/dental-insurance-checklist/compliancelog` — PHI access/change events and approval history.
