# React Implementation — Dental Insurance Checklist

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for dental offices.
- `src/views/ChecklistDemo.tsx` — Demo procedure-specific checklist.
- `src/views/ReworkCalculator.tsx` — Calculator for verification admin rework.
- `src/views/OfficeAuditForm.tsx` — Office workflow audit form.
- `src/views/PilotSignup.tsx` — Controlled pilot signup page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
