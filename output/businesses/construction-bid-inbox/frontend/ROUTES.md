# Frontend Routes

- `/construction-bid-inbox/landing` — Public page for turning chaotic bid requests into a clean bid pipeline.
- `/construction-bid-inbox/bidinboxdemo` — Show an email/RFP transformed into structured scope and checklist.
- `/construction-bid-inbox/pipelinecalculator` — Estimate admin hours saved and bid throughput increase.
- `/construction-bid-inbox/tradefitform` — Collect trade, region, project size and current estimating workflow.
- `/construction-bid-inbox/pilotsignup` — Invite contractors to a manual bid-inbox cleanup pilot.
