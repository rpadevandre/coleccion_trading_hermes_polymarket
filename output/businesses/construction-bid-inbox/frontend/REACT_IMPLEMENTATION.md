# React Implementation — Construction Bid Inbox

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for contractors.
- `src/views/BidInboxDemo.tsx` — Demo bid request extraction.
- `src/views/PipelineCalculator.tsx` — Calculator for admin hours and bid throughput.
- `src/views/TradeFitForm.tsx` — Trade/geography fit intake form.
- `src/views/PilotSignup.tsx` — Pilot signup page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
