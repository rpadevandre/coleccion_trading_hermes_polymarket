# React Implementation — Chiropractic Reactivation Engine

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for chiropractic clinics.
- `src/views/RecoveryCalculator.tsx` — Calculator for recovered appointments.
- `src/views/CampaignPreview.tsx` — Demo segment and message preview.
- `src/views/InactiveListAuditForm.tsx` — Inactive list audit form.
- `src/views/PilotSignup.tsx` — Pilot signup page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
