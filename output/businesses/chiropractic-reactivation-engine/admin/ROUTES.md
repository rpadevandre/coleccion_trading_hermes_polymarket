# Admin Routes

- `/admin/chiropractic-reactivation-engine/reactivationdashboard` — Segments, scheduled messages, replies and recovered appointments.
- `/admin/chiropractic-reactivation-engine/segmentbuilder` — Build and review inactive patient cohorts.
- `/admin/chiropractic-reactivation-engine/campaignapproval` — Review/edit/approve message drafts before sending.
- `/admin/chiropractic-reactivation-engine/replyinbox` — Track patient replies, booked, no response, opt-out.
- `/admin/chiropractic-reactivation-engine/compliancelog` — Data import, consent, opt-out and approval audit events.
