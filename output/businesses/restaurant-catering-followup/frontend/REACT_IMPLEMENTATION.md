# React Implementation — Restaurant Catering Follow-Up Copilot

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for restaurants.
- `src/views/EventInquiryDemo.tsx` — Demo inquiry parsing.
- `src/views/CateringRevenueCalculator.tsx` — Calculator for lost catering value.
- `src/views/InboxAuditForm.tsx` — Inbox workflow audit form.
- `src/views/PilotSignup.tsx` — Pilot signup page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
