# React Implementation — Property Maintenance Triage

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation landing page for property managers.
- `src/views/TenantRequestDemo.tsx` — Demo request intake and triage result.
- `src/views/ROICalculator.tsx` — Calculator for manager time saved and emergency leakage.
- `src/views/PortfolioAuditForm.tsx` — Audit form for unit count and workflow pain.
- `src/views/PilotSignup.tsx` — Pilot booking page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
