# Admin Views — Property Maintenance Triage

## 1. Triage inbox

Columns:

- Request ID.
- Property/unit.
- Tenant initials or anonymized contact.
- Category.
- Urgency.
- Safety flag.
- Completeness score.
- Age / SLA timer.
- Assigned coordinator.
- Status.

Filters:

- Emergency.
- Urgent.
- Incomplete.
- Awaiting tenant.
- Vendor-ready.
- Duplicate candidate.

## 2. Request detail

Panels:

- Original tenant message and uploaded media.
- AI summary with confidence.
- Emergency/safety rule hits.
- Missing information.
- Tenant response draft.
- Vendor work-order draft.
- Timeline/audit log.
- Internal notes.

Actions:

- Approve tenant follow-up.
- Edit summary.
- Change urgency.
- Assign vendor category.
- Mark duplicate.
- Export work order.
- Close request.

## 3. Vendor routing board

- Preferred vendors by property and trade.
- Availability notes.
- License/insurance status placeholder.
- After-hours contact rules.
- Last routed jobs.

## 4. Property/unit directory

- Property name.
- Unit label.
- Region/timezone.
- Warranty/appliance notes.
- Preferred vendors.
- Entry instructions.
- Emergency escalation contact.

## 5. Rules and templates

Admin-editable but versioned:

- Emergency keywords.
- Category taxonomy.
- Tenant clarification templates.
- Vendor work-order templates.
- SLA thresholds.
- Disclosure text for AI-assisted triage.

## 6. Analytics

- Requests/month.
- Minutes saved estimate.
- Response-time trend.
- First-touch completeness.
- Top issue categories.
- AI override rate.
- Vendor-ready conversion rate.

## 7. Audit and compliance screen

- Search all events by request ID.
- Export request history.
- View who approved each outgoing message.
- Confirm no unapproved AI message was sent.
