# Property Maintenance Triage

**Score:** 71/90

## English-first positioning

Turn vague tenant maintenance texts into clean work orders.

## Spanish adaptation

Convierte mensajes vagos de mantenimiento en órdenes de trabajo claras.

## Target buyer

Small property managers, landlords with 20-300 doors, small multifamily operators.

## Target user

Tenant, property manager, maintenance coordinator, approved vendors.

## Problem

Tenants send vague maintenance texts at all hours; managers waste time classifying urgency, requesting photos, routing vendors and documenting response.

## Offer v0

AI maintenance intake and triage inbox: tenant form/SMS, photo prompts, urgency scoring, vendor message drafts and audit logs.
