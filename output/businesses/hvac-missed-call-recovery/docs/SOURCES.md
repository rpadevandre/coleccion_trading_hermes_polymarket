# Sources — HVAC Missed-Call Recovery

Web research used during the 2026-06-08 enrichment pass. Treat the statistics as validation clues, not audited claims for customer-facing copy until verified against primary sources or pilot data.

- https://pcnanswers.com/missed-call-revenue-study/ — Missed-call revenue framing; references consumer abandonment after unanswered calls and caller behavior.
- https://agentzap.ai/blog/home-services-phone-statistics — Home-services phone statistics and missed-call value framing.
- https://www.callrail.com/blog/missed-call-statistics — Call tracking / missed-call context for local service businesses.
- https://www.homeadvisor.com/cost/heating-and-cooling/repair-an-ac-unit/ — HVAC job-value context useful for ROI assumptions.

## How to use these sources internally

- Use source-backed language in validation scripts as a hypothesis: “many emergency callers keep dialing until someone answers.”
- Do not promise a fixed recovered-revenue number before collecting shop-specific call logs.
- Ask prospects for the last 30 days of missed-call count, average ticket, and booking rate so the ROI model uses their own numbers.
