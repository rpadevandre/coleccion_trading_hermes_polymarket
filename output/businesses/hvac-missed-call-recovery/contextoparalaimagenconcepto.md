# Contexto para la imagen concepto — hvac-missed-call-recovery

> Guía para generar imágenes conceptuales, hero backgrounds, mockups y placeholders coherentes con el branding del negocio.

Fecha: 2026-06-10

## 1. Descripción del producto

**Producto:** HVAC Missed-Call Recovery

AI-assisted intake, urgency triage, owner alerts and audit-ready follow-up for small HVAC operators who lose revenue after hours.

**Headline principal:** Turn every missed HVAC call into a reviewed dispatch opportunity.

**Buyer/contexto:** Built for owner-led HVAC teams with 2–25 technicians and no 24/7 dispatcher.

## 2. Branding / dirección visual

urgent but trustworthy home-service operations: cold-air blue, dispatch control, orange revenue urgency

Debe sentirse como producto B2B vendible, no como scaffold ni plantilla genérica.

## 3. Paleta de colores

- **Primary:** `#2563EB` — CTAs principales y estados activos.
- **Secondary:** `#0F172A` — estructura, headings, navegación, dashboards.
- **Accent:** `#f97316` — highlights, urgencia, revenue, badges.
- **Background:** `#F8FAFC` — fondo SaaS limpio.
- **Surface:** `#FFFFFF` — cards, paneles, formularios.
- **Text:** `#0F172A` — texto principal.
- **Muted:** `#64748B` — texto secundario.
- **Border:** `#D8E0EA` — separadores.

## 4. Concepto visual recomendado

HVAC dispatch command center, service van, missed-call queue, cold blue gradients, orange urgency accents.

## 5. Composición recomendada

- Formato principal: `16:9`.
- Tamaño sugerido: `1920x1080`.
- Dejar espacio negativo para texto overlay.
- Elemento principal centro-derecha o fondo medio.
- Usar overlays SaaS/dashboards sutiles.
- No usar texto legible dentro de la imagen.
- No usar logos reales ni datos reales.

## 6. Qué evitar

- Stock photo genérica sin relación con el producto.
- Texto incrustado en la imagen.
- Logos reales o marcas registradas.
- Datos personales/clientes reales.
- Estética low-budget o plantilla genérica.
- Paletas ajenas al `DESIGN.md`.

## 7. Prompt base sugerido

```text
HVAC dispatch command center, service van, missed-call queue, cold blue gradients, orange urgency accents, polished B2B SaaS hero background, colors aligned with primary #2563EB, secondary #0F172A, accent #f97316, safe negative space for text overlay, realistic but premium, no readable text, no logos, no real customer data.
```

## 8. Relación con otros archivos

- `REPO_DESCRIPTION.md` explica el rol del negocio dentro de `output/businesses/`.
- `DESIGN.md` define tokens exactos de UI.
- `IMAGE_REQUIREMENTS.md` define lista de imágenes, tamaños y prioridades.
- `frontend/` y `admin/` deben usar esta dirección visual para placeholders y futuros assets.
