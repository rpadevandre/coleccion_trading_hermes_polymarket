# Admin Routes

- `/admin/hvac-missed-call-recovery/opsdashboard` — KPIs: missed calls captured, urgent jobs, recovered revenue, response time.
- `/admin/hvac-missed-call-recovery/callqueue` — Review all inbound call items with AI classification and confidence.
- `/admin/hvac-missed-call-recovery/calldetail` — Transcript, extracted fields, dispatch decision, edit history and notes.
- `/admin/hvac-missed-call-recovery/routingrules` — Configure service areas, emergency categories and notification recipients.
- `/admin/hvac-missed-call-recovery/analytics` — Weekly recovery trends, lead source breakdown and missed opportunity value.
