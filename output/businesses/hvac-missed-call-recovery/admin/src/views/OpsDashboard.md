# Admin View — OpsDashboard

KPIs: missed calls captured, urgent jobs, recovered revenue, response time.

## Operators

- owner/admin
- staff/reviewer
- auditor/compliance role where relevant

## Actions

- view
- filter
- edit/review
- approve/reject
- export/report where safe

## Guardrails

- AI suggestions are labeled as suggestions
