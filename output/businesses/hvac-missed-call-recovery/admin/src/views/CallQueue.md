# Admin View — CallQueue

## Purpose

Review all inbound call items with AI classification and confidence.

## Operators

- owner/admin
- staff/reviewer
- auditor/compliance role where relevant

## Actions

- view
- filter
- edit/review
- approve/reject
- export/report where safe

## Guardrails

- destructive actions require confirmation
- AI suggestions are labeled as suggestions
- all status/rule changes are audit logged
