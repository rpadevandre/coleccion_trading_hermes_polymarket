# React Implementation — Med Spa Lead Recovery

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for med spas.
- `src/views/LeadLeakCalculator.tsx` — Calculator for lost consult value.
- `src/views/InstagramDMFlow.tsx` — Demo DM/form/call to consult flow.
- `src/views/FreeLeadAuditForm.tsx` — Lead workflow audit form.
- `src/views/PilotSignup.tsx` — Pilot conversion page.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
