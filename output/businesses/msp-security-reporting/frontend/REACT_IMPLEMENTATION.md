# React Implementation — MSP Security Reporting Copilot

    This frontend is a Vite + React + TypeScript prototype scaffold for internal incubation.

    ## Views implemented

    - `src/views/Landing.tsx` — Public validation page for MSPs.
- `src/views/ReportBeforeAfter.tsx` — Raw technical noise vs executive report.
- `src/views/ClientProofDemo.tsx` — Demo client report approval flow.
- `src/views/StackAuditForm.tsx` — Tool stack and reporting pain intake.
- `src/views/PilotSignup.tsx` — Pilot signup on anonymized report sample.

    ## Local run after repo split

    ```bash
    npm install
    npm run dev
    ```

    ## Current limitation

    This is intentionally local/prototype code under `output/businesses`. It is not deployed and has no production API connection yet.
