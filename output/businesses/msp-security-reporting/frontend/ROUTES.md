# Frontend Routes

- `/msp-security-reporting/landing` — Public page for MSPs turning security work into client-ready reports.
- `/msp-security-reporting/reportbeforeafter` — Show raw security noise vs executive summary/report.
- `/msp-security-reporting/clientproofdemo` — Demonstrate monthly client report generation and approval flow.
- `/msp-security-reporting/stackauditform` — Collect tools used, number of clients and reporting pain.
- `/msp-security-reporting/pilotsignup` — Invite MSPs to test on one anonymized client report.
